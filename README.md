# mcalc

A program for computing measure scores

## How to use

# Install
```
$ git clone https://github.com/wtkm11/mcalc
$ python3 -m venv venv
$ source venv/bin/activate
$ pip install -e .
```

# Run
`$ python ./cli.py measure MEASURE_NAME` where `MEASURE_NAME` is the name of a
measure like `AMI` or `COPD`
